# Block 
分为三种block

## NSGlobalBlock 全局
## NSMallocBlock   堆
## NSStackBlock 栈

```
- (void)viewDidLoad {
    [super viewDidLoad];
    [self blocktest];
}

- (void)blocktest{
    int a = 10;
    void (^block)(void) = ^{
        NSLog(@"block");
    };
    block();
    
    NSLog(@"%@",block);
    NSLog(@"%@",^{
        NSLog(@"%d",a);
    });
}
```
 只要block添加上 = 号，栈block被改成堆block

修改外部变量
```
1. _ _block 把观察到的变量由栈区copy到堆.
- (void)blocktest{
   __block int a = 10;
    NSLog(@"前%p",&a);
    void (^block)(void) = ^{
        a = 11; 
        NSLog(@"中%p",&a);
    };
    block();
    NSLog(@"后%p",&a);
}
 _ _block 是传地址不是传值
```
