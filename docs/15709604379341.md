# UI离屏渲染
## 在屏渲染
意为当前屏幕渲染,指的是GPU的渲染操作是在当前用于显示的屏幕缓冲区中进行
## 离屏渲染
意为离屏渲染，指的是GPU在当前屏幕缓冲区以外新开辟的一个缓冲区进行渲染操作

指定UI视图的属性在合成之前不能再屏幕显示的时候，会出现屏幕渲染。屏幕渲染和GPU有关，指的是GPU在当前屏幕缓冲区以外新开辟的一个缓冲区进行渲染操作。


### 何时会触发离屏渲染
* 圆角（当和maskToBounds一起使用时）
* 图层蒙版
* 阴影
* 光栅化

### 为何要避免
会引发GPU 离屏渲染会有额外的开销
CPU + GPU 处理时间超过16.7ms,导致UI掉帧
